# PWA with Vite tool

https://vite-plugin-pwa.netlify.app/

## Basic React app

Create the project scaffolding:

    npm init vite
    cd vite-pwa-lab/
    pnpm install

Basic test:

    pnpm dev

## PWA

    pnpm i vite-plugin-pwa -D
    pnpm i workbox-window -D
    pnpm i @rollup/plugin-replace -D
